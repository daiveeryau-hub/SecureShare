<h1>🔐 SecureShare - Secure File Sharing Platform</h1>
<p>SecureShare is a secure file-sharing web application built using Python Flask. It encrypts uploaded files using AES-256 encryption before storing them on the server.</p>

<h2>🚀 Features</h2>
<ul>
<li>AES-256 File Encryption</li>
<li>Password-Protected Downloads</li>
<li>Expiring Download Links</li>
<li>Maximum Download Limit</li>
<li>Secure Dashboard</li>
<li>File Upload & Download</li>
<li>Automatic Expired File Cleanup</li>
<li>Responsive UI</li>
</ul>

<h2>🛠 Technologies Used</h2>
<ul>
<li>Python</li>
<li>Flask</li>
<li>SQLite</li>
<li>HTML5</li>
<li>CSS3</li>
<li>JavaScript</li>
<li>AES-256 Encryption</li>
</ul>
